Materi: Error Handling di Java

Selamat! Kamu telah menguasai konsep Enkapsulasi. Sekarang mari pelajari bagaimana menangani kesalahan dalam program.

1. Mengenal Jenis Exception (Error)

Dalam pemrograman Java, kesalahan atau *exception* secara garis besar dibagi menjadi tiga kategori utama:

Syntax Error

Kesalahan tata bahasa kode.

System.out.printl("Hi") // Kurang huruf 'n'

Runtime Error

Kesalahan saat program berjalan.

int x = 10 / 0; // ArithmeticException

Logic Error

Program jalan, tapi output salah.

luas = panjang + lebar; // Harusnya dikali

2. Struktur Try-Catch

Blok try digunakan untuk menaruh kode yang berpotensi error, sedangkan catch menangkap dan menangani error tersebut.

try {
  // Kode yang mungkin error
  int hasil = 10 / 0;
} catch (ArithmeticException e) {
  // Menangani error
  System.out.println("Terjadi kesalahan: Tidak bisa membagi dengan nol!");
}
